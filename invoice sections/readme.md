### Create a new invoice section

https://learn.microsoft.com/en-us/azure/cost-management-billing/manage/mca-section-invoice 

To create an invoice section, you need to be a billing profile owner or a billing profile contributor. For more information, see Manage invoice sections for billing profile.

1. Sign in to the Azure portal.

2. Search for Cost Management + Billing.

![image](https://github.com/OktaySavdi/Azure/assets/3519706/b91cc8bc-d3a6-458e-8c73-56bfeb482e08)

3. Select Billing profiles from the left-hand pane. From the list, select a billing profile. The new section will be displayed on the selected billing profile's invoice.

![image](https://github.com/OktaySavdi/Azure/assets/3519706/1d358e82-f13f-42e3-bf0b-1d706b908634)

4. Select Invoice sections from the left-hand pane and then select Add from the top of the page.

![image](https://github.com/OktaySavdi/Azure/assets/3519706/d72a802a-511d-4f88-ad1a-2eae77355879)

5. Enter a name for the invoice section.

![image](https://github.com/OktaySavdi/Azure/assets/3519706/ecb0c3c9-03d0-42cc-a9f9-6031713058fa)


az billing invoice section create - https://learn.microsoft.com/en-us/cli/azure/billing/invoice/section?view=azure-cli-latest
